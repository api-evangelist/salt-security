Salt Security launched Salt Code, a new product that embeds security policies directly into AI coding assistants like Claude, Cursor, and GitHub Copilot, eliminating the gap between company security standards and AI-generated code. The core problem is that AI assistants have no awareness of security wikis or compliance documentation, yet now produce nearly half of all code including APIs and MCP server infrastructure. Salt Code shifts security enforcement upstream to code creation time, ensuring generated code is compliant automatically without requiring developers to remember security requirements.
